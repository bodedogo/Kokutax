<h1 align="center"> KokuTax</h1>

#### KokuTax é uma aplicação web que simula o cálculo de impostos para trabalhadores, com base no salário mensal e nas principais deduções aplicáveis.

O projeto foi desenvolvido com foco em aprendizado prático de backend com Flask, integração com frontend e aplicação de lógica de programação em um cenário real.

---

🚀 Funcionalidades

- Inserção de salário mensal
- Cálculo automático de:
  - Salário anual
  - Dedução salarial (Employment Income Deduction)
  - Previdência
  - Seguro de saúde
  - Dedução básica
  - Renda tributável
  - Imposto nacional
  - Imposto residencial
  - Seguro desemprego

---

🧠 Como funciona

O sistema segue o seguinte fluxo:

1. O usuário insere o salário mensal no formulário
2. O formulário envia os dados para o backend (Flask)
3. O Python realiza todos os cálculos
4. Os resultados são retornados para o HTML
5. A página é renderizada novamente com os valores atualizados

▶️ Como rodar o projeto

1. Clone o repositório:

git clone https://github.com/bodedogo/Kokutax

2. Entre na pasta:

cd Kokutax

3. Instale o Flask (se necessário):

pip install flask

4. Execute o projeto:

python app.py

5. Acesse no navegador:

http://127.0.0.1:5000/
